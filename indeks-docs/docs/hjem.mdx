## Bruk Indeks med AI-assistenter

Hele dokumentasjonen er tilgjengelig som ren tekst som kan limes direkte inn i Claude, ChatGPT, Cursor eller andre LLM-verktøy:

- <a href="/llms-full.txt"><code>llms-full.txt</code></a> — hele dokumentasjonen i én fil, klar til å limes inn som kontekst
- <a href="/llms.txt"><code>llms.txt</code></a> — hierarkisk indeks hvis du vil at AI-en skal slå opp enkeltsider

Hver dokumentasjonsside finnes også som `.md`-variant — legg `.md` etter URL-en, for eksempel <a href="/docs/komponenter/textfield.md"><code>/docs/komponenter/textfield.md</code></a>.
