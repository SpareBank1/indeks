# Stack

Stack er en layout-primitiv for å stable innhold i én dimensjon — vertikalt eller horisontalt. Den fungerer både som custom element (`<ix-stack>`) og som CSS-klasse (`.ix-stack`).

## Egnet til

- Stable seksjoner, kort eller elementer loddrett med jevnt mellomrom
- Plassere ikoner, etiketter og knapper side om side horisontalt
- Bygge opp sammensatte layout-strukturer ved å neste stacks

## Uegnet til

- Todimensjonal layout — bruk CSS Grid og [grid-utilities](../../utility-klasser/oversikt#grid)
- Komplekst innholdsoppsett som artikler og dashboard-views

## Kom i gang

`ix-stack` har `md` gap som standard. Bruk `gap`-attributtet (eller prop) for å overstyre.

## Universell utforming

### Hva du selv må sørge for

- **Bruk semantiske elementer når konteksten krever det** — `ix-stack` er et ukjent HTML-element uten semantisk rolle. Nettleseren behandler det som generisk blokkinnhold, på samme måte som `<div>`. Bruk `as`-prop i React (f.eks. `<HStack as="nav">`) eller CSS-klasser direkte på et native element når innholdet krever semantikk.

På native elementer (ikke `<ix-stack>`) virker **ikke** attributter — bruk modifier-klasser:

```html
<!-- CSS-klasse på semantisk element — modifier-klasser styrer retning og gap -->
<section class="ix-stack ix-gap-md">
  <h2>Seksjonstittel</h2>
  <p>Innhold her.</p>
</section>

<!-- Tilsvarende med <ix-stack> — her bruker du attributter i stedet for klasser -->
<ix-stack gap="md">...</ix-stack>
```

### `<ix-stack>` — attributter

Attributter kan bare brukes på `<ix-stack>`-elementet, ikke på vanlige `<div>`-er.

| Attributt | Verdi | Beskrivelse |
|-----------|-------|-------------|
| `horizontal` | *(ingen verdi)* eller `"center"` | Horisontal retning, barn midtstilt |
| `horizontal` | `"start"` | Horisontal retning, barn toppjustert |
| `horizontal` | `"end"` | Horisontal retning, barn bunnjustert |
| `vertical` | *(ingen verdi)* | Vertikal retning, barn startjustert (eksplisitt) |
| `vertical` | `"center"` | Vertikal retning, barn midtstilt |
| `vertical` | `"end"` | Vertikal retning, barn bunnjustert |
| `gap` | `none` \| `2xs` \| `xs` \| `sm` \| `md` \| `lg` \| `xl` \| `2xl` \| `3xl` \| `4xl` | Avstand mellom barn. Standard: `md` |
| `reverse` | *(ingen verdi)* | Reverserer retningen |
| `nowrap` | *(ingen verdi)* | Hindrer linjebryting |

### CSS-klasser

Klasser fungerer på både `<ix-stack>` og vanlige elementer som `<div>`.

| Klasse | Beskrivelse |
|--------|-------------|
| `ix-stack` | Vertikal stack, barn justert til start |
| `ix-stack-vertical` | Eksplisitt vertikal, tilsvarende `ix-stack` |
| `ix-stack-vertical-center` | Vertikal, barn midtstilt |
| `ix-stack-vertical-end` | Vertikal, barn høyrejustert |
| `ix-stack-horizontal` | Horisontal, barn midtstilt (center) |
| `ix-stack-horizontal-center` | Horisontal, barn midtstilt (eksplisitt) |
| `ix-stack-horizontal-start` | Horisontal, barn toppjustert |
| `ix-stack-horizontal-end` | Horisontal, barn bunnjustert |
| `ix-stack-reverse` | Reverserer retningen. Kombiner med retningsklasse |
| `ix-stack-nowrap` | Hindrer linjebryting |

Standard gap er `md`. Bruk `gap`-attributtet på `<ix-stack>` eller `ix-gap-*` utility-klasser på andre elementer for å overstyre.

## Tilpasning med CSS

Stack-komponentene har ingen egne CSS custom properties. Bruk `ix-justify-*` og `ix-items-*` utility-klasser for ytterligere tilpasning:

```html
<ix-stack horizontal gap="md" class="ix-justify-between">
  <div>Venstre</div>
  <div>Høyre</div>
</ix-stack>
```
